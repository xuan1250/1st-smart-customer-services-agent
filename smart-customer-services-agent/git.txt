git command for this project

first create project:
git init -> create project into clone local
git add . -> add all changed to clone local
git remote add origin https://.... -> remote add all file to repository in github 



after editted this file in project, let:
git add . -> add all file to git clone
git commit -m "what is editted in project?" -> decriptions what is changed
git push -u origin master/branch -> push all changed to repository


after eddited in laptop window and 
git push -u origin master/branch -> push to repository in github
git pull -> sync all code to macos
